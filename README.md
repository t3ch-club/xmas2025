## 環境セットアップ手順

### 1. Visual Studio Code と Docker をインストール・セットアップする

事前に以下をインストールしてください。

- [Visual Studio Code](https://code.visualstudio.com/)
- [Podman Desktop](https://podman-desktop.io/)

### 2. このリポジトリを手元にダウンロードする

以下のいずれかの方法でリポジトリを取得してください。

**方法A: Git clone を使う**
```bash
git clone https://github.com/t3ch-club/xmas2025
```

**方法B: GitHub から ZIP をダウンロード**
- リポジトリページの「Code」→「Download ZIP」からダウンロード
- ダウンロードした ZIP を展開

### 3. Anthropic API のトークンを取得・配置する

1. Slack で [吉田周平に DM を送信](https://oro.slack.com/team/UA3U3ULQ2)
2. Anthropic API のトークン発行を依頼
3. もらったトークンを `.claude/.key` に配置する

### 4. このリポジトリを VSCode の DevContainer で開く

1. VSCode でこのリポジトリのフォルダを開く
2. 左下の「><」アイコンをクリック（または `Cmd+Shift+P` / `Ctrl+Shift+P` でコマンドパレットを開く）
3. 「Dev Containers: Reopen in Container」を選択
4. コンテナのビルドと起動を待つ

### 5. Claude Code で好き放題開発を進める！

準備完了！あとは自由にアイデアを形にしていきましょう 🎄
